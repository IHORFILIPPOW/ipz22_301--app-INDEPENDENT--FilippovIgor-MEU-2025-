Самостійна Робота 1: Основи HTML

1. Що таке HTML?

HTML (HyperText Markup Language) — це основа для створення веб-сторінок, що дозволяє структурувати контент.

Ось базова структура HTML-документа:

<!DOCTYPE html>
<html>
  <head>
    <title>Назва веб-сторінки</title>
  </head>
  <body>
    <h1>Заголовок</h1>
    <p>Це приклад веб-сторінки.</p>
  </body>
</html>

2. HTML Редактори коду та IDE

HTML можна писати в будь-якому текстовому редакторі, проте спеціалізовані редактори (IDE) такі як Visual Studio Code, Atom, або Sublime Text значно полегшують процес написання коду завдяки підсвічуванню синтаксису.

3.HTML Основні приклади

Приклад простої HTML-сторінки, що містить заголовки, абзаци та списки:

<!DOCTYPE html>
<html>
  <body>
    <h1>Заголовок</h1>
    <p>Це приклад абзацу тексту.</p>
    <ul>
      <li>Елемент 1</li>
      <li>Елемент 2</li>
    </ul>
  </body>
</html>

4. HTML Елементи

HTML елементи складаються з відкриваючих і закриваючих тегів, між якими розміщується вміст. Наприклад:

<p>Це абзац.</p>
<a href="https://example.com">Це посилання</a>

5. HTML Атрибути

Атрибути надають додаткову інформацію про елементи. Наприклад, href вказує адресу для посилання, а alt — альтернативний текст для зображення:

<a href="https://example.com">Перейти</a>
<img src="image.jpg" alt="Опис зображення">

6. HTML Параграфи

Абзаци створюються за допомогою тегу <p>:

<p>Це приклад абзацу тексту.</p>

7. HTML Посилання

Посилання (елемент <a>) дозволяють переходити на інші веб-сторінки. Атрибут href задає URL:

<a href="https://www.spotify.com/ua-uk/free/">Перейти на Spotify</a>
Посилання на Spotify

8. HTML Зображення

Зображення вставляються за допомогою тегу <img> з атрибутами src та alt:

<img src="image.jpg" alt="Приклад зображення">

9. HTML Таблиці

Таблиці використовуються для організації даних у структурованій формі:

Назва Вартість
Молоко 35 грн
Кефір 38 грн
Сир 29 грн

10. Списки в HTML

HTML підтримує різні типи списків:

Невпорядкований список:

  • Елемент 1
  • Елемент 2
  • Елемент 3

Впорядкований список:

  1. Елемент 1
  2. Елемент 2
  3. Елемент 3
